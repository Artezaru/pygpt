Welcome to pygpt's documentation!
==================================

``pygpt`` is a Python package that provides tools for creating and managing interactive discussions powered by OpenAI's GPT models.

The API of the package is described in the :doc:`./api` section.

The package ``pygpt`` also includes various user-friendly tools for managing discussions, including:
- Creating, loading, and saving discussions.
- Setting system prompts and token limits.
- Searching and interacting with discussions.

A User Guide is available in the :doc:`./usage` section.

The list of the implemented commands is provided in the :doc:`./commands` section.
