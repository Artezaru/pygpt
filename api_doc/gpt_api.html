pygpt.GPTApi#

class pygpt.GPTApi(api_key: bytearray, model: str = 'gpt-3.5-turbo')[source]#

GPTApi is a class that provides an interface to communicate with OpenAI’s GPT models.

This class facilitates connecting to the OpenAI API using an API key, setting the desired GPT model, and sending messages to obtain responses from the model. The supported models are restricted to “gpt-3.5-turbo” and “gpt-4”.

Parameters:

  • api_key (bytearray) – The API key used to authenticate with the OpenAI API. It must be a bytearray.

  • model (str) – The name of the GPT model to be used. It must be one of the following: [“gpt-3.5-turbo”, “gpt-4”]. Default is “gpt-3.5-turbo”.
