API Reference
==============

The package ``pygpt`` is composed of the following functions, classes, and modules:

- ``pygpt.Discussion`` class provides tools to create, manage, and save discussions powered by OpenAI's GPT models.
- ``pygpt.GPTApi`` class handles interactions with the OpenAI API.
- ``pygpt.Message`` class allows constructing and managing conversation messages.
- ``pygpt.UserInterface`` class provides a terminal-based interface for managing discussions and interacting with GPT.
